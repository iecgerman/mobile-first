# mobile-first

Curso de Responsive Design: Maquetación Mobile First

En este curso aprenderás a diseñar y maquetar sitios web que se adapten perfectamente a cualquier dispositivo, comenzando por los móviles. A través de una combinación de teoría y práctica, dominarás técnicas modernas de Responsive Design, usarás herramientas como Figma y Lighthouse, y aplicarás estándares como BEM y media queries. Al final del curso, construirás un sitio completo desde su wireframe hasta su versión responsiva optimizada.
